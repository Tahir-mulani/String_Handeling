# String Handling & Practice Repository

A comprehensive collection of **string manipulation problems** solved using multiple approaches. This repository demonstrates solving string problems with both **manual logic** and **Java Collection Framework** (List, Set, Queue, Map).

## 📌 Overview

This repository contains daily practice code for string handling problems. Each problem is implemented in two ways:
- **Manual Logic**: Using basic string operations and loops
- **Collection Framework**: Using Java collections (HashSet, HashMap, ArrayList, etc.) for optimal solutions

 
 
## 📊 Comparison Table

| Approach | Time Complexity | Space Complexity | Best For |
|----------|-----------------|------------------|----------|
| Manual Logic | O(n²) | O(n) | Small strings, Learning concepts |
| Collection Framework (HashSet) | O(n) | O(n) | Production code, Large strings |
